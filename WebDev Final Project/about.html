About

Master Kim Il-Kwon's Taekwon-Do was first established in 1987 and
hopes to teach people from the ages of 11 to 25 the art of Taekwon-Do,
Master Il-Kwon came up with the idea after helping some tourists
from a mugging on a walk home one night. He had the idea of setting up
a class where he can teach his skills so that people wouldn't be as defenceless.

This is our dojo in Dublin and by far our most popular, hover the mouse over the
image to see the a picture of our champion match last year!

The plugin I used is called jQuery zoom by Jack Moore and can be found here
The plugin allows the website user to zoom into a picture by
hovering, dragging or clicking on the picture. I chose to use the grabbing function because it
looked nice with the grabbing icon and its can be annoying if the mouse automatically zooms
in if you hover over. The grab function gives the user a lot more contol.
I had to download two javascript files called jquery.zoom.js and jquery.zoom.min.js
The code I had to use included $('#img').zoom({on:'grab'}); I had run into a problem previously where
I had img, img1, img2, img3 all equate to the grab cursors but there was a bug so
I had to let them all equate individually.
